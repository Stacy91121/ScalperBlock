ScalperBlock — Give real customers a fair chance

ScalperBlock helps online sellers keep high-demand items available for real people, not bots or resellers. We watch checkout signals in real time and deliver an easy-to-understand risk score and a recommended action so merchants can protect inventory without interrupting genuine buyers.

Core Capabilities

Clear, fast scoring

Receive a simple risk score (0–100) and a suggested action (ALLOW, CHALLENGE, BLOCK) so you can respond automatically or review cases quickly.

Multiple signals combined

We blend IP reputation, request patterns, browser signals, and checkout timing to spot automated or suspicious activity while minimizing disruption to real users.

Easy management

Tune rules, review flagged transactions, and capture merchant feedback to improve decisions over time — all from a simple interface.

API Example —

Send a JSON transaction payload to get back a clear risk score and recommended action. The API is designed to be fast and easy to integrate; production use requires authentication (API key or JWT). Below is a sample request and a simulated response.

Request (JSON)

{
  "ip_address": "198.51.100.12",
  "user_agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64)...",
  "timestamp": 1699999999000,
  "product_id": "SKU-1234",
  "session_id": "sess-abc-001",
  "checkout_speed_ms": 1500
}

Sample Response

{
  "risk_score": 42,
  "recommended_action": "ALLOW",
  "breakdown": {
    "ip_score": 10,
    "velocity_score": 5,
    "fingerprint_score": 12,
    "behavior_score": 15
  }
}

Latency target: <100ms. Auth required for production.

Architecture & Diagrams

Example backend: Java + Spring Boot for the API and scoring logic, with PostgreSQL for data. The design includes modular scoring components (IP reputation, velocity, fingerprinting, behavior), an admin area for adjusting rules, and logging for auditing and improvement.

  • Class diagram: ApiEndpoint → RiskScoringEngine → reputation/velocity/fingerprint providers
  • DB: Users, Tasks, Items, Chats, Messages (relational)
